Start Gebäudebeschreibung

Auf dieser Website können Sie sich allozentrische Beschreibungen von Gebäuden in Textform ausgeben lassen. Ziel ist es, den Besuch eines (öffentlichen) Gebäudes im Vorfeld zu planen. Um dies zu erleichtern soll eine allgemeine allozentrische Gebäudebeschreibung zur Verfügung gestellt werden. Der Begriff "allozentrisch" bedeutet soviel wie positionsunabhängig. Das heißt, die Ausgaben werden, ausgehend von einem übergeordneten Bezugspunkt, bereitgestellt. Beispielsweise währen Beschreibungen wie "auf der Linken Seite..." egozentrisch, also nicht allozentrisch, da dieser Satz abhängig von der Position der Person ist. Auf der anderen Seite wäre "in Richtung Norden..." eine allozentrische und somit positionsunabhängige Beschreibung.

In den Beispiel-Beschreibungen, die Sie erhalten können, wenn Sie auf den Tab "Gebäudebeschreibung" klicken, werden eben solche allozentrischen Beschreibungen erstellt. Die Gebäudebeschreibungen wurden in jeweils drei Teile aufgeteilt: eine allgemeine Gebäudebeschreibung, eine Stockwerkbeschreibung und pro Stockwerk jeweils Beschreibungen zu den einzelnen Gängen beziehungsweise Foyers. Hierfür wurde ein neues Koordinatensystem über jedes Stockwerk gelegt, mit dem Mittelpunkt vor dem Punkt, an dem man das Stockwerk betritt. In den Beispielen ist das jeweils der Aufzug. Von diesem Punkt aus werden die Beschreibungen erstellt, genutzt wurden dafür Uhrzeitangaben (12 Uhr für Norden, 3 Uhr für Osten etc.) in Abhängigkeit des Koordinatensystems. So müssen sich die Angaben nicht mit den tatsächlichen Himmelsrichtungen decken. Mit dem zweiten Tab "Gebäudebeschreibung" (oben in der Navigationsleiste) kommen Sie zur Seite mit den Gebäudebeschreibungen. Dort finden Sie einen Button "Info", der ein Popup-Fenster öffnet. Auf diesem Fenster finden Sie nähere Informationen zu den Grammatiken. Also Informationen, wie die Ausgaben zu interpretieren sind, wie die Grammatik aufgebaut ist, etc. Unter dem Info-Button finden Sie drei Auswahlmöglichkeiten (hier Picker). Die erste Auswahl legt das Gebäude fest (zwei zur Auswahl), die zweite legt das Stockwerk fest (zwei pro Gebäude) und die dritte legt den Gang oder das Foyer fest (beliebige, aber feste Auswahl; abhängig vom Stockwerk). Nach jeder Auswahl können Sie sich entscheiden die Beschreibung zu spezialisieren oder die Auswahl ausgeben zu lassen. Für die Ausgabe müssen Sie nur den Button "Beschreibung starten" klicken. Dieser Button befindet sich unter den drei Pickern. Die Text-Beschreibung befindet sich unterhalb des Buttons "Beschreibung starten". Ein Beispiel: Sie wählen Gebäude 1 aus. Jetzt können Sie sich die allgemeine Gebäudebeschreibung ausgeben lassen (Button klicken) oder Sie wählen zusätzlich Stockwerk 2 aus. Nun haben Sie die Möglichkeit sich die Stockwerkbeschreibung für das zweite Stockwerk des ersten Gebäudes anzeigen zu lassen oder Sie wählen einen der Gänge in diesem Stockwerk aus. Sie haben zu jedem Zeitpunkt die Wahl. Wenn Sie nun doch die Gebäudebeschreibung haben wollen, klicken Sie entweder bei der Stockwerkauswahl auf "Default" (die erste Wahl) oder nochmal auf Gebäude 1. Dies setzt Ihre Stockwerk- und Gangauswahl beide auf "Default". Anschließend können Sie sich die Beschreibung mit einem Klick auf den Button "Beschreibung starten" ausgeben lassen.

Sie können nun auf die zweite Seite gehen. Klicken Sie hierfür auf den Tab "Gebäudebeschreibung".
